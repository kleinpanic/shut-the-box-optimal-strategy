# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅        |

## Reporting a Vulnerability

This is a static client-side application with no backend. If you find a security issue (e.g., XSS in the rendered HTML), please open a GitHub issue labeled `security`.

For sensitive findings, email the maintainer directly before disclosing publicly.
